Config¶

class mosaic.types.config.Config(content=None, defaults=None)[source]¶

Bases: mosaic.types.struct.Struct

A Config is a specific type of (mutable and not extensible) Struct that populates itself with some provided default values. This allows to have some default configuration that can then be superseded by the user.

Parameters

  • content (dict-like, optional) – Dict-like object to initialise the Struct, defaults to empty.

  • defaults (dict-like, optional) – Dict-like object that provides the default values of the Config object.
