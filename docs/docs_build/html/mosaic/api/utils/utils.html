mosaic.utils.utils.memory_limit()[source]¶

Get the memory limit (in bytes) for this system.

Takes the minimum value from the following locations: - Total system host memory - Cgroups limit (if set) - RSS rlimit (if set)

Returns

Memory limit.

Return type

float
